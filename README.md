# 🛡️ Cadence Biometric Core

A localized, dual-layer biometric security system featuring real-time facial recognition and Machine Learning-based keystroke dynamics. 

Designed for offline, cross-hardware compatibility utilizing a One-Class Support Vector Machine (SVM) to bypass standard AVX2 CPU instruction limitations.

## ⚙️ Prerequisites
* **Python 3.10, 3.11, or 3.12** installed on your system.
* A functional webcam for the Face ID hardware hook.

## 🚀 Quick Start Guide

Follow these steps exactly to build the environment and launch the dashboard without dependency conflicts.

### 1. Clone the Repository
Download or clone this folder to your local machine, then open a terminal inside the root directory (`cadence offline system`).

### 2. Create the Virtual Environment
Do not install these packages to your global Python. Create an isolated environment:

**Windows:**
```cmd
python -m venv venv
venv\Scripts\activate

after venv activates
python -m pip install --upgrade pip
pip install -r requirements.txt
'''
